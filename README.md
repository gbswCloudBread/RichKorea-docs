## 구현 형태
* 공공의 이익을 위해 지역 홍보 서비스
* 로드밸런싱을 활용한 고 과용성 서비스
* 서버 장애 시 빠른 대처를 위한 모니터링 구축
* 설명이 불 필요한 UI
